# HTML / text URLs the scraper will try in addition to hkjunkcall.com.
#
# Format: one URL per line. Blank lines and lines starting with '#' are
# ignored. Anything after '#' on the same line is treated as a comment.
#
# For each URL the scraper:
#   1. Fetches the page with a realistic Chrome User-Agent.
#   2. Strips HTML tags to flat text.
#   3. Extracts every 8-digit HK phone number ([2-9]\d{7}).
#   4. Looks at a 60-char window around each match for a category hint
#      (EN + 繁中 keywords). Anything without a hint becomes "other".
#
# A URL that returns 403 / times out / yields 0 numbers is logged and
# silently dropped — never breaks the build. The build report lists
# every URL we tried and how many entries each contributed.
#
# Add new sources here without touching Python.

# ── HK Police Anti-Deception Coordination Centre (ADCC) ───────────────

# pages publish narrative warnings, not lists. Kept here on the chance
# they ever post a "known scam numbers" table.

https://www.police.gov.hk/ppp_tc/04_crime_matters/adcc/
