Common questions

How do I know files never leave my device?

Open Chrome DevTools (F12) → Network tab → use any tool → watch: zero outbound requests to any server. The processing runs entirely in your browser using JavaScript. We have no server to receive your files even if we wanted to.

What happens to my data if I cancel Pro?

Your data stays on your device. We don't hold it. Downgrading to Free means Pro features become unavailable, but nothing is deleted from our side — because we never had it.

Do the 28 mobile apps also work locally?

Yes. All apps are designed for local-first operation. Core features work offline. Where a network request is required (e.g. breach database lookup), it is explicitly disclosed in the app.

Is there a refund policy?

30-day no-questions refund on all plans. And if we ever receive your files from any tool, the full year is refunded automatically.
